Jeu Flatcraft


Technologies utilisées :


Java

IntelliJ IDEA

Gradle

GitLab


Description du projet :

En équipe de 4, nous avons réalisé un jeu Flatcraft en Java qui reprend le concept de Minecraft mais en 2D. Nous pouvons miner des ressources mais également sauter, creuser et fabriquer des objets à partir de ressources collectées.

Nous avons utilisé le pattern Modèle Vue Contrôleur et de nombreux patrons de conception pour optimiser le code. Nous avons également utilisé Gradle pour faire du projet un fichier exécutable en une commande.

Pokédex en ligne

Technologies utilisées :


Python

Flask

PyCharm

GitLab


Toujours en groupe, nous avons développé un site web de Pokédex en utilisant Python avec le framework Flask et en intégrant l'API PokéAPI. L'objectif principal du site est de permettre aux utilisateurs de rechercher un Pokémon en saisissant son nom dans un formulaire. En utilisant Flask, nous avons créé des routes pour gérer la recherche des Pokémon dans l'API PokéAPI. Une fois que l'utilisateur soumet le formulaire, le backend envoie une requête à l'API, récupère les données JSON associées au Pokémon recherché, puis les affiche de manière lisible sur le site. Nous avons accordé une attention particulière à l'interface utilisateur pour la rendre conviviale, avec un design attractif.

Le site affiche les caractéristiques du Pokémon, telles que son type, ses statistiques et ses capacités, offrant ainsi une expérience agréable aux utilisateurs.

Site web de gestion de scènes

Technologies utilisées :


PHP

Laravel

PHPStorm

GitLab


En groupe, nous avons développé un site web utilisant le framework Laravel. Ce site offre des fonctionnalités avancées de gestion de scènes, avec une possibilité de s'authentifier. L'authentification est une composante clé de notre site, permettant aux utilisateurs de créer des comptes, de se connecter et d'accéder à leur profil. Les utilisateurs connectés peuvent créer leurs propres scènes, les supprimer et les modifier. L'adinistrateur peut, quant à lui, gérer toutes les scènes. Les utilisateurs peuvent également ajouter des commentaires à des scènes et les supprimer.

Pour garantir une expérience utilisateur optimale, nous avons accordé une attention particulière au design en utilisant le framework CSS TailWindCSS. L'utilisation du framework Laravel a permis de gérer la base de données ainsi que l'authentification.

Projet marathon web


Technologies utilisées :


Java

IntelliJ IDEA

Gradle

GitLab


Description du projet :

Nous avons récemment participé à un projet de marathon web organisé par l'IUT de Lens. En tant qu'équipe de 4 étudiants en BUT informatique, notre mission était de développer le backend du projet. En parallèle, nos collègues en BUT MMI se sont occupés du frontend. Ce projet, réalisé en PHP avec le framework Laravel, nous a permis d'approfondir nos compétences dans le développement web, en mettant en œuvre des concepts avancés de backend. La collaboration interdisciplinaire a enrichi notre expérience et renforcé nos compétences en tant qu'étudiants en informatique.

Dans ce projet nous devions réaliser un site web permettant de gérer des histoires. Nous pouvons nous authentifier, créer nos propres histoires, les supprimer, créer des chapitres dans les histoires, ajouter des commentaires. Le but était de pouvoir créer des histoires dont nous somes le héros, où l'on peut choisir la suite sous forme de questions réponses.

Ray Tracing

Technologies utilisées :


Java

IntelliJ IDEA

Gradle

GitLab


Description du projet :

Il s'agit d'un projet de groupe où la consigne était la suivante : implémenter en Java une bibliothèque proposant un lanceur de rayons (ray tracing). Notre bibliothèque doit permettre de prendre en entrée un fichier texte décrivant une image (une scène), et de produire l'image correspondante comme vous pouvez voir ci-dessus. Nous avons veillé à appliquer les bonnes pratique de développement, et en particulier de la programmation orientée objet, au cours du développement de cette bibliothèque. Le ray tracing permet de générer par ordinateur des images avec un effet 3D, en simulant le parcours de la lumière au sein du’une scène, dans laquelle se trouvent des objets qui sont éclairées par cette lumière.

Ce projet a permis de mettre en oeuvre mes compétences en Java, mais également en travail d'équipe. Nous communiquons sans cesse entre coéquipiers pour connaître l'avancement du projet, réaprtir les tâches restantes et optimiser notre temps. Ce projet conséquent a été personnellement très enrichissant.
