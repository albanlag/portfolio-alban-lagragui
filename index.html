Jeu Flatcraft

Technologies utilisées :

Java

GitLab

IntelliJ Idea

Gradle

Description du projet :

En équipe de 4, nous avons réalisé un jeu Flatcraft en Java qui reprend le concept de Minecraft mais en 2D. Nous pouvons miner des ressources mais également sauter, creuser et fabriquer des objets à partir de ressources collectées.

Nous avons utilisé le pattern Modèle Vue Contrôleur et de nombreux patrons de conception pour optimiser le code. Nous avons également utilisé Gradle pour faire du projet un fichier exécutable en une commande.

Pokédex en ligne

Technologies utilisées :

Python

Flask

Pycharm

GitLab

Description du projet :

Toujours en groupe, nous avons développé un site web de Pokédex en utilisant Python avec le framework Flask et en intégrant l'API PokéAPI. L'objectif principal du site est de permettre aux utilisateurs de rechercher un Pokémon en saisissant son nom dans un formulaire. En utilisant Flask, nous avons créé des routes pour gérer la recherche des Pokémon dans l'API PokéAPI. Une fois que l'utilisateur soumet le formulaire, le backend envoie une requête à l'API, récupère les données JSON associées au Pokémon recherché, puis les affiche de manière lisible sur le site. Nous avons accordé une attention particulière à l'interface utilisateur pour la rendre conviviale, avec un design attractif.

Le site affiche les caractéristiques du Pokémon, telles que son type, ses statistiques et ses capacités, offrant ainsi une expérience agréable aux utilisateurs.

Site web de gestion de scènes

Technologies utilisées :

PHP

Laravel

PHPStorm

GitLab

Description du projet :

En groupe, nous avons développé un site web utilisant le framework Laravel. Ce site offre des fonctionnalités avancées de gestion de scènes, avec une possibilité de s'authentifier. L'authentification est une composante clé de notre site, permettant aux utilisateurs de créer des comptes, de se connecter et d'accéder à leur profil. Les utilisateurs connectés peuvent créer leurs propres scènes, les supprimer et les modifier. L'adinistrateur peut, quant à lui, gérer toutes les scènes. Les utilisateurs peuvent également ajouter des commentaires à des scènes et les supprimer.

Pour garantir une expérience utilisateur optimale, nous avons accordé une attention particulière au design en utilisant le framework CSS TailWindCSS. L'utilisation du framework Laravel a permis de gérer la base de données ainsi que l'authentification.
